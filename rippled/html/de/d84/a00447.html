Detailed Description

template<class Derived>
class beast::asio::enable_wait_for_async< Derived >

Facilitates blocking until no completion handlers are remaining. If Derived has this member function:

void on_wait_for_async (void)

Then it will be called every time the number of pending completion handlers transitions to zero from a non-zero value. The call is made while holding the internal mutex.
