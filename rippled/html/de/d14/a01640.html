TER ripple::core::Taker::process ( Amounts const &  flow,
Offer const &  offer 
)
inline

Process the result of fee and funds calculation on the offer.

To protect the ledger, conditions which should never occur are checked. If the invariants are broken, the processing fails.

If processing succeeds, the funds are distributed to the taker, owner, and issuers.

Returns
false if processing failed (due to math errors).
