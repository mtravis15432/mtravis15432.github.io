Error beast::sqdb::session::open ( String  fileName,
std::string  options = "timeout=infinite|mode=create|threads=multi" 
)

Open a database.

The database at the specified path is opened. The connection string is a set of "{key}={value}" pairs separated with the pipe symbol ('|'). Choices for key and value are:

timeout = (number) || "infinite"
mode = "read" || "write" || "create"
cache = "shared" || "private"
threads = "single" || "multi"
