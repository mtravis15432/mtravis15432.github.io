Detailed Description

template<class Duration, bool IsSteady = true>
class beast::manual_clock< Duration, IsSteady >

Manual clock implementation. This concrete class implements the abstract_clock interface and allows the time to be advanced manually, mainly for the purpose of providing a clock in unit tests.

Template Parameters
Thelength of time, in seconds, corresponding to one tick.
