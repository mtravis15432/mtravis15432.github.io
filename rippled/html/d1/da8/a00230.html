Detailed Description

template<class T, class Tag = buffer_view_default_tag>
class beast::buffer_view< T, Tag >

A view into a range of contiguous container elements.

The size of the view is determined at the time of construction. This tries to emulate the interface of std::vector as closely as possible, with the constraint that the size of the container cannot be changed.

Template Parameters
TThe underlying element type. If T is const, member functions which can modify elements are removed from the interface.
TagA type used to prevent two views with the same T from being comparable or assignable.
