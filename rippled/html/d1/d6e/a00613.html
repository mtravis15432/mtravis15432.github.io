std::size_t beast::asio::HandshakeDetectLogicSSL2::max_needed ( )
inlinevirtual

How many bytes maximum we might need.

This is the largest number of bytes that the detector might need in order to come to a conclusion about whether or not the handshake is a match. Depending on the data, it could come to that conclusion sooner though.

Use read_some instead of read so that the detect logic can reject the handshake sooner if possible.
