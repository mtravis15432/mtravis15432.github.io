Detailed Description

template<class Int, class Tag>
class beast::tagged_integer< Int, Tag >

A type-safe wrap around standard unsigned integral types

The tag is used to implement type safety, catching mismatched types at compile time. Multiple instantiations wrapping the same underlying integral type are distinct types (distinguished by tag) and will not interoperate.

A tagged_integer supports all the comparison operators that are available for the underlying integral type. It only supports a subset of arithmetic operators, restricting mutation to only safe and meaningful types.
