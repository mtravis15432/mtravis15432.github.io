Detailed Description

Adapts a ServiceQueue to dispatch Checker handler completions. This lets the Logic have its Checker handler get dispatched on the ServiceQueue instead of an io_service thread. Otherwise, Logic would need a ServiceQueue to dispatch from its handler.
