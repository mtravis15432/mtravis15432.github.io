Detailed Description

template<class Key, class Hash = std::hash <Key>, class KeyEqual = std::equal_to <Key>, class Allocator = std::allocator <Key>>
class ripple::CycledSet< Key, Hash, KeyEqual, Allocator >

Cycled set of unique keys. This provides a system of remembering a set of keys, with aging. Two containers are kept. When one container fills, the other is cleared and a swap is performed. A key is considered present if it is in either container.
