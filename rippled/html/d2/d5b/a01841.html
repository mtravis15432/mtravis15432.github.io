Detailed Description

template<class Handler, class Context>
class beast::asio::detail::wrapped_handler< Handler, Context >

A handler which wraps another handler using a specfic context. The handler is invoked with the same io_service execution guarantees as the provided context.

Note
A copy of Context is made.
