22 // Run some of the tests registered by the TEST() macro. If the
23 // environment variable "ROCKSDB_TESTS" is not set, runs all tests.
24 // Otherwise, runs only the tests whose name contains the value of
25 // "ROCKSDB_TESTS" as a substring. E.g., suppose the tests are:
26 // TEST(Foo, Hello) { ... }
27 // TEST(Foo, World) { ... }
28 // ROCKSDB_TESTS=Hello will run the first test
29 // ROCKSDB_TESTS=o will run both tests
30 // ROCKSDB_TESTS=Junk will run no tests
31 //
