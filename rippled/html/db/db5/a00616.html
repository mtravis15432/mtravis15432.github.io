Detailed Description

template<typename Stream, typename Logic>
class beast::asio::HandshakeDetectorType< Stream, Logic >

A wrapper to decode the handshake data on a Stream.

Stream must meet these requirements

For detect: SyncReadStream

For async_detect: AsyncReadStream

Logic must meet this requirement: HandshakeDetectLogic
