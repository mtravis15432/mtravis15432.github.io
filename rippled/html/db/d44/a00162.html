Detailed Description

template<class TrivialClock>
class beast::basic_seconds_clock< TrivialClock >

A clock whose minimum resolution is one second. The purpose of this class is to optimize the performance of the now() member function call. It uses a dedicated thread that wakes up at least once per second to sample the requested trivial clock.
