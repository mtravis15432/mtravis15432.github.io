456 // Internal form:
457 // 1: If amount is zero, then value is zero and offset is -100
458 // 2: Otherwise:
459 // legal offset range is -96 to +80 inclusive
460 // value range is 10^15 to (10^16 - 1) inclusive
461 // amount = value * [10 ^ offset]
462 
463 // Wire form:
464 // High 8 bits are (offset+142), legal range is, 80 to 22 inclusive
465 // Low 56 bits are value, legal range is 10^15 to (10^16 - 1) inclusive
