43  // Attempt to open the table that is stored in bytes [0..file_size)
44  // of "file", and read the metadata entries necessary to allow
45  // retrieving data from the table.
46  //
47  // If successful, returns ok and sets "*table_reader" to the newly opened
48  // table. The client should delete "*table_reader" when no longer needed.
49  // If there was an error while initializing the table, sets "*table_reader"
50  // to nullptr and returns a non-ok status.

116  // Read the block, either from sst file or from cache. This method will try
117  // to read from cache only when block_cache is set or ReadOption doesn't
118  // explicitly prohibit storage IO.
119  //
120  // If the block is read from cache, the statistics for cache miss/hit of the
121  // the given type of block will be updated. User can specify
122  // `block_cache_miss_ticker` and `block_cache_hit_ticker` for the statistics
123  // update.
