Detailed Description

template<std::size_t Bytes>
class beast::UnsignedInteger< Bytes >

Represents a set of bits of fixed size.

The data is stored in "canonical" format which is network (big endian) byte order, most significant byte first.

In this implementation the pointer to the beginning of the canonical format may not be aligned.
