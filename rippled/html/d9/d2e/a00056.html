Detailed Description

template<typename Value, class SharedMutexType = SharedMutexAdapter <RecursiveMutex>>
class beast::SharedData< Value, SharedMutexType >::Access

Provides non-const access to the contents of a SharedData. This acquires a unique lock on the underlying mutex.
