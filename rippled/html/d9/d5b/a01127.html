Detailed Description

template<class Stream>
class beast::asio::PrefilledReadStream< Stream >

Front-ends a stream with a provided block of data.

When read operations are performed on this object, bytes will first be returned from the buffer provided on construction. When those bytes are exhausted, read operations will then pass through to the underlying stream.

Write operations are all simply passed through.
